Open Source Travel Agent

把攻略、车次、酒店、预算和分享页收束成一条可复用的旅游规划链路。

这一版聚焦中国境内旅行规划。你可以直接在网页里跑，也可以复用同一套后端给 Codex skill、CLI 和自动化分享链路使用。

查看云南案例分享页 查看案例详情

Current Scope

  • 中国境内
  • 铁路 / 自驾
  • 登录态复用
  • 分享快照
  • 不代下单

浏览器优先级固定为 `Edge -> 项目持久化 Chromium -> Chrome`。

Sources

当前接入

  • 小红书攻略摘要与 POI
  • 12306 只读车次查询
  • 高德路线估算与地图展示
  • 携程 / 美团 / 飞猪酒店候选
  • 动态 share + Excel / PDF 导出

Boundaries

能力边界

  • 做规划，不做代下单、代抢票或绕过官方限制
  • 不要求导出 Cookie，只复用本机已有登录态
  • 验证失败后支持 checkpoint 恢复，而不是整单重跑
  • 分享页读取快照，不会在外部访问时重新抓站

Publish Ready

开源发布前检查

仓库里已经内置发布前脱敏检查脚本，会扫描本机绝对路径、局域网 IP、临时隧道域名、疑似密钥和不该被跟踪的导出文件。

推荐每次 push 前执行 
python scripts/prepublish_check.py
