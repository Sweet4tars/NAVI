# 云南自驾案例方案（修正版）

## 基本约束

- 出行时间：`2026-04-30` 晚宜宾集合，`2026-05-05` 晚回宜宾解散
- 成员：
  - 小毛：成都出发
  - 小唐：重庆出发
  - 小张：重庆出发
  - 小熊：宜宾出发
- 车辆：车在宜宾，统一从宜宾出发
- 出行方式：`1 台车` 自驾
- 目标主线：`宜宾 -> 西昌 -> 丽江 -> 大理 -> 攀枝花 -> 宜宾`
- `昭通`：只保留为可选缓冲点，本次 `5 天 5 晚` 主线里不默认并入

## 这版和上一版的关键修正

- 不再走 `昆明` 主线
- 主线改成你指定的 `宜宾 -> 西昌 -> 丽江 -> 大理 -> 攀枝花 -> 宜宾`
- 酒店筛选逻辑改成：
  - `先定住宿位置，再选该位置里最有性价比的酒店/民宿`
  - `地理位置优先`
  - `单间目标价尽量控制在 ¥160-250`
  - `周边吃饭方便 / 停车方便 / 去景点方便`
  - 可接受 `经济连锁 / 舒适型酒店 / 预算友好民宿`
- 餐饮规则改成：
  - 每一顿给 `2 个候选`
  - 同一天 `早餐 / 午餐 / 晚餐` 不重复同一家
  - 优先选 `大众点评 / 美团 / 小红书` 上高分、评论量较多、地方特色明确的店

## 酒店预算说明

- 你给的硬约束我先按更严格的口径执行：`2 人 1 间，尽量不超过 ¥250/晚`
- 但这次是 `五一` 档期，动态房价很可能上浮
- 所以下面的酒店候选理解为：
  - `优先选平日/常规档常见在 250 内的酒店或民宿`
  - 真正下单前，仍需要在 OTA 页面再确认当晚房型是否还压得住 `¥250`
- 这也是为什么我在部分城市会优先给出：
  - `汉庭 / 7 天 / 星程 / 城市便捷 / 预算型本地酒店`
  - 而不是高景观海景房

## 集合与发车原则

### 4 月 30 日晚集合

- 集合地建议：`宜宾西站` 或 `当晚入住酒店`
- 集合时间建议：`20:00-21:00`
- 分工建议：
  - 小毛：优先成都到宜宾的高铁
  - 小唐、小张：优先重庆到宜宾的高铁或大巴
  - 小熊：负责车辆、补给、接人和行李整合
- 当晚必须在宜宾住 1 晚，不建议直接夜开去西昌

### 驾驶原则

- `5 月 2 日` 和 `5 月 5 日` 是这条线最累的两个驾驶日
- 每 `2-2.5 小时` 必须休息一次
- 最后一段进城路尽量不叠加景点打卡
- 停车便利优先级高于景观

## 住宿位置优先逻辑

这版住宿不再先看“哪家酒店最出名”，而是先看“住在哪片最合理”。

### 这条线的优先顺序

1. `离吃饭近`
   - 晚上到店后步行或短打车就能解决晚饭
   - 早餐不用为了一口吃的再额外折腾
2. `离停车场近 / 酒店自身停车方便`
   - 节假日自驾最怕停车难
   - 能停好车，比海景和网红装修更重要
3. `离玩的地方近，但不要近到吵`
   - 最理想的是：景点外围 5-15 分钟可达，不住进最拥挤的核心区
4. `在这个位置里选最便宜、最稳、适合学生住的店`

### 各城市先住哪儿

- `宜宾`：优先 `宜宾西站周边`，其次 `合江门 / 大观楼`
  - 原因：集合接人最方便，夜里也更容易解决吃饭和停车
- `西昌`：优先 `航天大道 / 旅游集散中心`，其次 `建昌古城外围`
  - 原因：去邛海、月亮湖和市区美食街都不算远，停车和次日出城效率更好
- `丽江`：优先 `古城南门 / 忠义市场`，其次 `大水车外围`
  - 原因：南门一带对自驾最友好，吃饭便宜，打车和停车都比住进古城深处强
- `大理`：优先 `古城南门 / 苍山门外围`，其次 `大理站周边`
  - 原因：古城外围比古城核心区更适合拖行李、停车和找吃的，且夜里更安静
- `攀枝花`：优先 `东区炳草岗 / 五十四转盘`
  - 原因：这是攀枝花最成熟的城区，吃饭、补给、停车和第二天一早返程都更方便

## 推荐主线

`宜宾 -> 西昌 -> 丽江 -> 大理 -> 攀枝花 -> 宜宾`

### 为什么这么排

- 在 `4 月 30 日晚集合 / 5 月 5 日晚回宜宾` 这个窗口里，这是包含 `西昌 + 丽江 + 大理 + 攀枝花` 的最稳妥走法
- `昭通` 如果强行并入，会把这条线从“偏紧”变成“过紧”
- 所以这版把 `昭通` 只当作：
  - 延长 1 晚时的备选
  - 或者必须规避长途疲劳时的应急缓冲点
